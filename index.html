HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An HTML attribute defines additional information about an element.
  • The title element defines the title for the webpage.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The body element defines the body of the document, which contains the content that is visible to the user.
  • The div element created a division in the page.
  • The span element creates an inline division.
  • The h1 - h6 elements respesent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unorderes lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on the webpage.
  • The br element creates a line of empty space, or a line break, between the two blocks of content.
  • The a element creates links to the same webpage or other webpages.
  • The section element defines a section in a document that contains thematic content.

CSS

  • Inline CSS is used when a developer wants to style an element directly in the HTML file.
  • Using the style tag, CSS ruls can be contained in the internal style sheet in the HTML file.
  • An external CSS style sheet is used to keep all CSS rules in a separate file.
  • Selectors define the element or attribute to which rules apple.
  • Declaractions contain the CSS property and the value of the CSS property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS box model states that each box in HTML constists of margins, borders, padding, and the content.
  • Class arttributes are used as selectors to style sections of code.
